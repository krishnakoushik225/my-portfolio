## Portfolio Features

| Feature | Implementation |
|---|---|
| Flicker-free dark/light mode | Blocking inline script reads `localStorage` before first paint, stamps `class="dark"` on `<html>` |
| Active nav on scroll | `IntersectionObserver` on all sections, updates nav highlight in real time |
| Scroll progress bar | 2px brand-purple gradient fixed to top of viewport via `window.scroll` listener |
| Mobile menu | Framer Motion `x: "100%"` → `x: 0` slide-in drawer with body scroll lock |
| Image fallback | `onError` hides broken `<img>` and shows a purple-indigo gradient placeholder |
| Open Graph previews | Rich link cards on LinkedIn, Slack, Discord via `og:image` + `og:description` |
| JSON-LD Person schema | Google knowledge graph indexing via `application/ld+json` in `<head>` |
| Sitemap + robots | Auto-generated at `/sitemap.xml` and `/robots.txt` via Next.js App Router |
| Keyboard accessibility | Brand-purple `focus-visible` ring injected globally via `useEffect` style tag |
| Tab filter animation | `AnimatePresence` fade-swap when switching skill domain filter |

## Getting Started Locally

```bash
# 1. Clone
git clone https://github.com/krishnakoushik225/portfolio.git
cd portfolio

# 2. Install
npm install

# 3. Run
npm run dev
# → http://localhost:3000

# 4. Build for production
npm run build && npm run start
```

No environment variables required. Fully static — no backend, no database, no API keys.
